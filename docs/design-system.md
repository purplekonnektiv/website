# PurpleKonnektiv Design System

PurpleKonnektiv uses a bold editorial layout inspired by community media sites, translated into a purple Nostr-native identity.

## Colors

- Purple Core: `#6D28D9`
- Deep Violet: `#241232`
- Electric Lilac: `#A855F7`
- Signal Pink: `#E879F9`
- Ink: `#151019`
- Paper: `#F7F2FF`
- Chalk: `#FFFDF7`
- Protocol Green: `#17A673`
- Relay Yellow: `#F7C948`

Purple should lead the identity, while off-white surfaces and dark ink preserve readability. Green and yellow are reserved for live, upcoming, or relay-status details.

## Layout

The homepage is a single-page experience with anchored sections:

- Navigation
- Ticker
- Hero
- Mission
- Feed
- Calendar
- Docs
- Footer

Repeated content uses rectangular cards with strong borders and offset shadows. Page sections are full-width bands rather than nested cards.

## Accessibility

- Maintain WCAG 2.1 AA contrast.
- Preserve visible focus states.
- Respect `prefers-reduced-motion`.
- Keep event-sourced images decorative unless meaningful alt text is available.
